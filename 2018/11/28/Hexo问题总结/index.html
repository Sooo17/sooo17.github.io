Hexo搭Blog的总结

发表于 2018-11-28

流程

  1. 新建repository
  2. 装必备包
    • node.js 通过nvm安装
    • hexo 通过npm安装
    • Git 通过homebrew安装
  3. 初始化blog
    • hexo init <~/>
  4. 新建文章
    • _post内添加.md文件
  5. 测试&发布
    • hexo s预览
    • hexo g生成文件
    • hexo d 发布
  6. 更新
    • sudo hexo clean && sudo hexo g && sudo hexo d
